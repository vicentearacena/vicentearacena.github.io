8- Creación de molde

Pasos

En primer lugar, hacemos un nuevo sketch, con una línea de simetría que servirá como referencia. Luego hacemos una superficie cerrada en el plano frontal para dar la forma del molde. Vamos al comando Revolve con el cual hacemos, seleccionamos la superficie y se realiza un molde redondo.

Hacemos unos fillet en los bordes para dar una forma mas fina al molde. Hacemos un nuevo sketch seleccionando la cara inferior y realizamos un cuadrado desde el centro. Extruimos el cuadrado y lo combinamos con la figura huella para que el área de esta quede en la extrusión.

Para finalizar realizamos un ángulo de desmolde para que no queden ángulos rectos, debido a que, la pieza puede ser difícil de desmoldar. Por último, la inspeccionamos para ver la composición del molde para poder imprimirlo en 3D y realizar varias copias.
